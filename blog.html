FAQ

1. What is the Document Object Model (DOM)?

In JavaScript, the Document Object Model (DOM) is an API that represents the structure of an HTML or XML document as a tree of objects, allowing JavaScript to interact with, manipulate, and update the document dynamically.

2.How do you select an element from the DOM?

  1. document.getElementsByClassName()
  2. document.getElementsByTagName()
  3. document.querySelector()
  4. document.querySelectorAll()

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in which a single event listener is added to a parent element rather than multiple event listeners to individual child elements.
This is useful because: It reduces the number of event listeners, improving performance, specially when dealing with many elements.

4. How do you manipulate an element's attributes and styles using the DOM?

Attributes:

1. element.setAttribute()
2. element.getAttribute()
3. element.removeAttribute()

Styles:

element.classList.add()
element.classList.remove()
element.classList.toggle()
